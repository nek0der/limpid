# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be respectful. Assume good intent. Keep discussions focused on the work.

To report a Code of Conduct concern (harassment, abusive behavior, etc.), open a [private Discussion](https://github.com/nek0der/limpid/discussions) marked confidential or contact the maintainer directly via the email on their [GitHub profile](https://github.com/nek0der). Security vulnerabilities have a separate channel — see [SECURITY.md](SECURITY.md). I'll respond within a few days.
